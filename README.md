# water-meter
Photoelectronic Sensor connected to a WeMos D1 Mini for collecting consumption data of an analog water meter.


## Part List
* Arduino (WeMos) D1 Mini        6€
* Buck Converter (15V to 5V)     2€
* Junction box                   2€
* Wago Connectors (3x)           4€
* Power Supply 15V               2€
* Barrel Connector               1€
* Resistor 10k Ω                 2€
* some cables                    5€
* Sunx EX-26A                  >25€

Total                          ~50€
